ipython-osx-menu
================

![Alt text](http://labs.radiantmachines.com/ipython.png "In action")

![Alt text](http://labs.radiantmachines.com/notebook.png "Open in browser")

Easily spawn and the access IPython notebook from the OS X Menu bar. 

This is super alpha.

just run it with 

``python menu.py``

##Requirements

* PyObjC
